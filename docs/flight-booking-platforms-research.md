## 二、各平台核心特点

### 携程
- **优势**：资源覆盖最全（国内+国际）；7×24人工客服；退改签处理能力最强；「价格曲线」功能可预判涨跌；机酒套餐性价比高
- **劣势**：日常裸价偏贵；界面营销弹窗多；默认不含行李额需注意
- **用户定位**：兜底选择，服务稳定，出了问题好解决

### 去哪儿
- **优势**：比价功能最强，聚合 30+ 代理商报价；「低价提醒」实用；尾单/凌晨航班折扣票多
- **劣势**：代理票源混杂，出票慢/舱位不符风险；低价票多为「不可退改」；**大数据杀熟问题被用户广泛反映**（刷新一次价格立涨 200+，2945 条评论讨论此问题）
- **用户定位**：比价入口，但需警惕动态调价

### 飞猪
- **优势**：支付宝积分抵现 + 88VIP 折扣 + 学生认证低价三重叠加；花呗分期免息；与南航/东航会员日叠加积分；促销活动频繁（双11/618）
- **劣势**：基础票价与携程持平，优惠需凑单；客服依赖智能客服；国际航线资源弱；存在大数据杀熟反映
- **用户定位**：薅羊毛首选，适合有阿里生态会员权益的用户

### 同程旅行
- **优势**：微信流量专属券多；「低价预约」「机票盲盒」「周三超惠省」等特色入口；AI 出行方案；下沉市场覆盖广
- **劣势**：国际航线深度不及携程；高端资源偏少
- **用户定位**：微信用户的自然选择，小城市航线覆盖好

### 智行
- **优势**：火车+机票一体化查询，适合组合出行
- **劣势**：用户反映同样存在价格虚高问题
- **用户定位**：高铁+飞机混合出行场景

### 航旅纵横
- **优势**：中国民航信息网络官方出品，航班动态最权威；汇聚 37+ 航司直销比价；准点率/值机信息最准
- **劣势**：机票购买非核心业务；价格优惠力度弱
- **用户定位**：航班追踪 + 探索官方直销价

### 天巡（Skyscanner）
- **优势**：全球比价最强，聚合上千家航司+OTA；「整月最低价」「目的地随意」模糊搜索；价格地图直观
- **劣势**：不直接出票，需跳转第三方；售后由出票方负责
- **用户定位**：国际机票比价的起点

### 航司官网
- **优势**：会员日全网最低价；积分兑换；无中间商退改更顺畅
- **劣势**：需要分别登录各航司，操作分散
- **用户定位**：常旅客的终极省钱手段

## 三、用户比价行为习惯

**标准流程（用户口诀）**：
> 天巡/谷歌航班比价 → 携程/飞猪二次议价 → 航司官网查会员价 → 叠加平台优惠券

**核心行为模式**：
- 多平台交叉验价是主流，没有人只用一个平台
- 清空浏览器记录 + 换设备查价，实测差价可达 385 元（防大数据杀熟）
- 关注「低价提醒」功能，设置心理价位等通知
- 周二下午/周三刷新价格，周五晚价格最高
- 起飞前 48 小时蹲守尾舱清仓价
- 国内线提前 15-30 天购买性价比最高
- 往返票比单程便宜约 40%，夜间航班便宜约 30%

## 四、场景化平台选择

| 场景 | 推荐平台 | 原因 |
|------|---------|------|
| 国内机票·价格优先 | 去哪儿 + 飞猪 | 比价 + 返现叠加 |
| 国内机票·服务优先 | 携程 | 售后最稳，退改最快 |
| 国际机票·比价 | 天巡 → 同程/携程下单 | 先比价再下单 |
| 国际机票·常旅客 | 飞猪（叠加航司会员积分）| 积分权益最大化 |

| 商务出差 | 携程企业账号 | 月结 + 发票 + 延误险 |
| 家庭亲子游 | 携程（机酒套餐）| 一站式 + 无忧退改 |
| 航班动态追踪 | 航旅纵横 | 官方数据最准 |

| 提前购（1-2 个月）| 航司官网会员日 | 会员日全网最低 |
| 微信用户/小众路线 | 同程旅行 | 微信专属券 + 百元机票入口 |

## 五、关键洞察

1. **没有绝对最便宜的平台**，多平台比价是用户共识，不是少数人的技巧
2. **大数据杀熟是用户最大痛点**，去哪儿、携程、飞猪、智行均被点名，用户已形成反制意识（换设备、清缓存）
3. **小红书用户画像偏年轻**，飞猪（阿里生态）和去哪儿（低价）在该群体中热度高
4. **携程**虽然价格不是最低，但因服务稳定仍是用户「兜底选择」
5. **航旅纵横**正在被更多用户发现，「突然发现航旅纵横的机票更便宜」类帖子获得高赞
6. **航司官网直购**被视为「终极省钱」手段，尤其是会员日场景
7. **国际机票**场景下，天巡 + 同程/携程的组合策略被广泛推荐
